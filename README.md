## What is it? 
It is a web interface that simplifies the use of bitcannad from a web server.  
This interface is intended to replace the main functions of bitcann-qt.

## Require

 - Web server (vps or simple ubuntu) 
 - Bitcannad
 - NodeJs 
 - Npm
